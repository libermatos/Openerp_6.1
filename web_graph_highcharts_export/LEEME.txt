***************************************************************
           Web_graph_highcharts_export
***************************************************************

Módulo para OpenERP 6.1 que permite exportar a PNG o PDF las gráficas Highcharts
generadas en el módulo web_graph_highcharts. Ver ejemplos disponibles en el 
módulo demo_graph_highcharts


Características
---------------
- Contiene un módulo para Highcharts que permite el cambio visual y la petición requerida
  para realizar el exportar. Este módulo es basado sobre el original presente en Highcharts,
  realizado por Torstein Honsi. Si el desarrollador necesita incluir o eliminar alguna opción
  del mismo puede valorar crear uno similar y referenciarlo en __openerp__.py o sentirse libre
  de modificar el proporcionado por el módulo.
- Contiene la traducción al idioma inglés y al español.


Instalación
-----------
Tiene dependencia con pycairo

* Windows
- Extraer el contenido del archivo: web_graph_highcharts_export\common\cairo.zip
  en una localización válida de PYTHONPATH. En Windows un ejemplo pudiera ser:
  C:\Python27\Lib\site-packages

* Linux
- Se encuentran disponibles en los repositorios


Documentación
